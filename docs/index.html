DrumToGrid RVQ-CE Qualitative Examples

Each example takes a hard rhythmic input drum grid (shown as a family×16th-note piano-roll) and renders drum audio in the DAC PCA-72 subspace. For every clip we compare three signals: the diffusion model (RVQ-CE, 25 steps), the deterministic direct PCA regressor baseline, and the ground-truth DAC reconstruction — across 6 held-out validation clips spanning tempo 93–145 BPM. All audio and images are local relative assets, so the page runs as static content.
